# Project-A
 🛠️ Kali Linux & Wireshark Practicals
This repository contains practical implementations of basic commands in Kali Linux and network packet analysis using Wireshark. It is designed for beginners in cybersecurity and networking.

📚 Practicals Included
🔹 Practical 4: Basic Commands in Kali Linux
🎯 Objective
To perform basic file and directory operations in Kali Linux.
🧾 Commands Covered
pwd → Displays the current working directory path
ls → Lists all files and folders
cd → Changes directory
mkdir → Creates a new directory
touch → Creates a new file
rm → Deletes a file

▶️ Steps
1.Open Kali Linux terminal
2.Check current directory:
Bash
pwd
3.List files:
Bash
ls
4.Change directory:
Bash
cd Documents
5.Create a folder:
Bash
mkdir BBA_BA_31
6.Create a file:
Bash
touch students
7.Delete file:
Bash
rm students

✅ Output
Basic file and directory operations executed successfully.

🔹 Practical 8: Wireshark Packet Analysis
🎯 Objective
To capture and analyze network traffic.
🧾 Tools Used
Wireshark

▶️ Steps
1.Install Wireshark
2.Open the application
3.Select a network interface
4.Click Start Capture
5.Generate traffic (e.g., browsing websites)
6.Click Stop Capture
7.Analyze packets using filters like:
http
tcp

✅ Output
Network packets are captured and analyzed successfully.
